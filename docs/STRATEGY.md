## 핵심 차별화 포인트 (심사위원 설득 포인트)

1. **Gemma 4만 가능**: 이미지+오디오 동시 입력은 E4B 고유 기능
2. **오프라인 동작**: 농촌 인터넷 없는 환경 → E4B 온디바이스 실행
3. **언어 장벽 제거**: 스와힐리어, 힌디어, 벵골어 등 현지어 응답
4. **즉시 데모**: 사진+음성 → 30초 진단 → 심사위원 직접 체험
5. **공개 데이터**: PlantVillage CC0 라이선스, 재현 가능
