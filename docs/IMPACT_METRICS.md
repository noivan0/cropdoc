## 📊 임팩트 시나리오 분석

### 전제 조건 (공통)
- **기준 인구**: 전 세계 소농 5억 명
- **스마트폰 보급률**: 아프리카 43%, 남아시아 38%, 동남아 65%
- **추정 타겟 농민 (스마트폰 + 작물 재배)**: 약 **2.5억 명**
- **작물 손실 기여 요인**: 병해충이 전체 손실의 25~40%
- **조기 진단으로 손실 감소율**: 50~70% (문헌 기반 추정)

---

### 시나리오 A: 보수적 (Conservative)

**가정**
- CropDoc 도달 농민: 타겟의 **1%** = 250만 명
- 농민당 연간 소득: USD 800 (세계은행 기준 개도국 평균)
- 병해충으로 인한 연간 손실: 소득의 20% = USD 160/농민
- 진단 정확도로 인한 손실 감소: 30%

**결과**
```
절약되는 손실 = 250만 명 × USD 160 × 30%
              = USD 1억 2,000만 (연간)

영향 받는 인구 = 250만 농민 × 평균 4인 가족
              = 1,000만 명의 식량 안보 개선
```

| 지표 | 보수적 수치 |
|------|-----------|
| 도달 농민 | **250만 명** |
| 연간 경제적 절감 | **USD 1.2억** |
| 식량 안보 혜택 인구 | **1,000만 명** |
| 진단 정확도 향상 | 30% 손실 감소 |

---

### 시나리오 B: 낙관적 (Optimistic)

**가정**
- CropDoc 도달 농민: 타겟의 **10%** = 2,500만 명
- 바이럴 확산 (농민 커뮤니티 입소문)
- 진단 정확도: 99%+ (PlantVillage 기반 Fine-tuning)
- 조기 개입으로 손실 감소: 60%

**결과**
```
절약되는 손실 = 2,500만 명 × USD 160 × 60%
              = USD 24억 (연간)

식량 생산 증가 = 2,500만 농민 × 평균 1.5톤/ha × 10% 생산성 향상
              = 375만 톤의 추가 식량

탄소 효율 = 비료/농약 낭비 감소로 이산화탄소 배출 감소
```

| 지표 | 낙관적 수치 |
|------|-----------|
| 도달 농민 | **2,500만 명** |
| 연간 경제적 절감 | **USD 24억** |
| 추가 식량 생산 | **375만 톤** |
| 식량 안보 혜택 인구 | **1억 명** |
| 온실가스 감소 | 추정 50만 톤 CO₂eq |

---

### 시나리오 C: 해커톤 데모 임팩트 (즉각적)

**단 1번의 정확한 진단이 만드는 변화**
```
케냐 농민 아마라 케이스:
- 재배 면적: 0.5 헥타르 토마토
- 예상 수확량: 2톤 (USD 600)
- 잎마름병 조기 발견 전: 80% 손실 예상 = USD 480 손실
- CropDoc으로 조기 진단: 적정 처리로 20% 손실 = USD 120 손실
- 회피된 손실: USD 360 (연간 소득의 45%)
- 4인 가족 3개월치 식량 보존
```
